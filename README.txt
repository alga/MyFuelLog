=========
MyFuelLog
=========

This is an Android app that helps keep track of fuel expenses,
calculates fuel mileage (in l/100km) and shows graphs.

Home page
---------

You can find MyFuelLog at GitHub:

  https://github.com/alga/MyFuelLog
